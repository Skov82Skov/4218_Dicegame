# 4218 Sprint 1 Starter

## Included
- Name entry page
- Name validation
- Local player identity storage
- Lobby placeholder
- Simple Express server with `/health`

## Run

### 1. Install
```bash
npm install
```

### 2. Start server
```bash
npm run dev:server
```

### 3. Start client
```bash
npm run dev:client
```

Client: http://localhost:3000
Server: http://localhost:4000/health
